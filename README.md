# Use...
```bash
git clone https://github.com/rubyon/vue3-sinatra.git

cd vue3-sinatra

# Install packages
bundle install
yarn

# Dev
foreman start
# Front end : http://localhost:8080
# Back end : http://localhost:8081

# Build
yarn build
APP_ENV=production ruby app.rb

# Clean
yarn clean
```
